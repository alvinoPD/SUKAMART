# 🛒 Sukamart

**Sukamart** adalah aplikasi web jual beli sederhana yang dibuat untuk memudahkan pengguna dalam melihat produk, melakukan pemesanan, dan mengelola data produk melalui dashboard admin.

Aplikasi ini dibuat menggunakan **PHP, MySQL, HTML, CSS, dan JavaScript**.

---

## 📌 Fitur Utama

### 👤 User

* Melihat daftar produk
* Melihat detail produk
* Melakukan pemesanan produk
* Melihat informasi produk yang tersedia

### 🛠️ Admin

* Login admin
* Dashboard admin
* Menambah produk
* Mengedit produk
* Menghapus produk
* Upload gambar produk
* Melihat data pesanan
* Mengelola stok produk

---

## 🖥️ Teknologi yang Digunakan

* **HTML** → Struktur halaman web
* **CSS** → Desain tampilan
* **JavaScript** → Interaksi halaman
* **PHP** → Backend aplikasi
* **MySQL** → Database penyimpanan data

## ⚙️ Cara Menjalankan Project

1. Install **XAMPP / Laragon**
2. Clone repository ini

```bash
git clone https://github.com/username/sukamart.git
```

3. Pindahkan folder ke dalam:

```
xampp/htdocs/
```

4. Jalankan **Apache** dan **MySQL** di XAMPP

5. Import database ke **phpMyAdmin**

6. Buka browser dan akses:

```
http://localhost/sukamart
```
